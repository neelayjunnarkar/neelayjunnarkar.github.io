Falling Blocks

Posted on August 11, 2015 by Neelay Junnarkar
This is a game I first wrote 2 years ago in my freshman year as my first graphical program when I began learning Java. Uptil that point, I had only been doing command-line programs in C++. Since then, I have implemented this in other languages I have learned in which I have done graphics (just C++, actually).
The point of this game is to avoid the randomly-positioned and randomly-sized blocks which fall from above.

Controls: Keyboard Arrow Keys
