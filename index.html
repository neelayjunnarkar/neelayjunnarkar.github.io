Released Version 2 of NetLib
Posted on October 25, 2015 by Neelay Junnarkar

Last year, for my robotics team, I began learning csockets and networking so we could send information for processes such as vision and data plotting between computers. I wrote a system of classes that wrapped lower-level csocket functionality; this system allowed us to achieve what we wanted at the time.

However, I recently began overhauling the system with knowledge I have gained over the past year because the previous system was overly specific and not able to be used for anything else but for the tailor-made programs. This overhaul is far more modular and will allow all the functionality of the previous system while not restricting the functionality.

The code can be found at my Networking repository on Github. Check it out!
