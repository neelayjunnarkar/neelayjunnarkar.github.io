Keyboard Layouts
Posted on June 5, 2016 by Neelay Junnarkar
Personal keyboard layouts that give access to accents and tilde, as well as mathematical operators and symbols. Show more...

Never again shall you use '--' instead of '—'!

I recently created a keyboard layout (using Microsoft Keyboard Layout Creator 1.4) for Windows so that I could easily use non-english symbols. Primarily, I wanted to be able to type for my Spanish class without having to keep switching between the US English Keyboard and the Latin America Spanish Keyboard. I also added mathematical symbols such as π, λ, θ, ∑, and equality comparison symbols. Symbols are accessible using the alt-gr key. Some of the locations of symbols are based on how macs place the symbols, but I changed much. For example, in my kebyoard, one types the ́ (accent) after the letter instead of before.

To install this keyboard, go to this repository, download the zip file ('njkeyboa.zip') in the repository, and run the setup exe.

Released Version 2 of NetLib
Posted on October 25, 2015 by Neelay Junnarkar
Networking library written in C++ using c-sockets. Show more...

Last year, for my robotics team, I began learning csockets and networking so we could send information for processes such as vision and data plotting between computers. I wrote a system of classes that wrapped lower-level csocket functionality; this system allowed us to achieve what we wanted at the time.

However, I recently began overhauling the system with knowledge I have gained over the past year because the previous system was overly specific and not able to be used for anything else but for the tailor-made programs. This overhaul is far more modular and will allow all the functionality of the previous system while not restricting the functionality.

The code can be found at my Networking repository on Github. Check it out!
